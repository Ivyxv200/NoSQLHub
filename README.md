# NoSQLHub
> 一个基于 Node.js + Express + MongoDB + Redis 构建的 NoSQL 技术社区论坛系统。
NoSQLHub 是一个面向开发者的技术交流平台，用户可以发布技术文章、评论互动、点赞收藏、关注其他用户，并通过搜索、推荐系统发现感兴趣的内容。
项目采用前后端分离架构，后端基于 Express 提供 RESTful API，MongoDB 存储核心业务数据，Redis 用于缓存、高频数据统计以及排行榜维护，同时结合 Socket.IO 实现实时消息通知。

# 📌 功能介绍

## 👤 用户系统
实现完整用户生命周期管理：
- 用户注册
- 用户登录
- 密码加密存储
- 用户信息修改
- 用户头像管理
- 用户权限控制


## 📝 帖子系统
用户可以进行技术内容交流：
- 发布帖子
- 编辑帖子
- 删除帖子
- 查看帖子详情
- 标签分类
- 图片上传


## 💬 社区互动
支持用户之间的互动：
- 评论
- 点赞
- 收藏
- 用户关注
- 粉丝列表
- 浏览历史


## 🔍 搜索系统
实现多维度搜索：
- 帖子关键词搜索
- 用户搜索
- 搜索建议
- 热门搜索统计


## 🤖 推荐系统
根据用户行为进行个性化推荐：
- 浏览记录
- 点赞记录
- 收藏记录
- 标签偏好
